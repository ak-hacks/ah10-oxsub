# OxSub
The substitute rower marketplace for Oxford College Rowing      
https://ah10-oxsub.herokuapp.com

# Tech stack
* [React](https://facebook.github.io/react/)
* [Bootstrap](http://getbootstrap.com/)

## Downstream dependencies
* [OxSub API](#)
* [Auth0](https://auth0.com/)
